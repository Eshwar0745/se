# **9. Jenkins Pipeline Using Script ‚Äî Ultra-Short Steps**

### **1. Create a Pipeline Job**

### **2. Add Pipeline Script**

Pipeline ‚Üí Definition: **Pipeline script**
Paste a simple script:

```groovy
pipeline {
    agent any
    tools{
        maven 'MAVEN_HOME'
    }
    stages {
        stage('git repo & clean') {
            steps {
                bat "rmdir  /s /q xes || exit 0"
                bat "git clone https://github.com/Eshwar0745/xes.git"
                bat "mvn clean -f xes"
            }
        }
        stage('install') {
            steps {
                bat "mvn install -f xes"
            }
        }
        stage('test') {
            steps {
                bat "mvn test -f xes"
            }
        }
        stage('package') {
            steps {
                bat "mvn package -f xes"
            }
        }
    }
}
```
